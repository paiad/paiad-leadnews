# Paiad Leadnews 媒体端

Vue 3 + TypeScript 媒体管理前端

## 技术栈

Vue 3 · TypeScript · Vite · Element Plus · Pinia · Vue Router · Axios · UnoCSS

## 快速开始

```bash
npm install
npm run dev
npm run build
```

## 结构

```
src/
├── api           # 接口
├── components    # 组件
├── router        # 路由
├── stores        # 状态
└── views         # 页面
    ├── Home      # 首页
    ├── Login     # 登录
    ├── news      # 内容管理
    ├── material  # 素材库
    ├── channel   # 频道管理
    └── links     # 链接速览
```

## 功能

- 内容发布与管理
- 素材库管理
- 频道管理
- 数据概览

## 开发

本地: http://localhost:5173  
后端: 自动代理至网关
